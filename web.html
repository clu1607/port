Sketchpad

A midterm project for Web Programming class mainly focused on P5. A simple sketchpad is created using P5. Users are able to draw on the sketchpad. Similar to a drawing program, users are able to choose a color, enter a size stroke, clear or save the drawing they created. At the side are some example pictures for users if they want to follow along.

Tasty Eats

A food based website for my Web Production II class Final Project which was heavily focused on PHP and MYSQL. Main recipes change depending on the day of the week using PHP. Viewers are able to browse around the website looking for recipes and they can give ratings/feedback and/or if they would like to contact and the information they enter will be stored using database. Ocean Pollution

A design for a website that promotes saving and protecting the ocean. The website will provide information about the current situation of the ocean such as the plastic waste being discarded causing sea animals to suffer. The website is like a sea. As the user keeps scrolling, the colors will get darker as if they're going deeper into the water. The menu nav will turn into an icon on the top right so it will be easy to access and won't block the content.
