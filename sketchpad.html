sketchpad

A midterm project for Web Programming class mainly focused on P5. A simple sketchpad is created using P5. Users are able to draw on the sketchpad. Similar to a drawing program, users are able to choose a color, enter a size stroke, clear or save the drawing they created. At the side are some example pictures for users if they want to follow along.
